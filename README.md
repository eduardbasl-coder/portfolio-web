# 🌐 basldev.fun

Moderní osobní portfolio a projektový web zaměřený na:

- 💻 Programování
- 🌐 Web development
- 🔌 Elektroniku
- 🖥️ Homelabbing
- ⚡ Networking

Web je vytvořen čistě pomocí HTML, CSS a JavaScriptu bez frameworků.

# 📸 Náhled

## Hlavní funkce webu

✅ Moderní dark UI  
✅ Animované pozadí  
✅ Responzivní design  
✅ Hover animace  
✅ Scroll animace  
✅ Blog systém  
✅ Sekce projektů
